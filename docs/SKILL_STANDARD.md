## 一、遵循的规范标准

本项目遵循以下两个开放标准：

### 1.1 Claude Code Skills 规范

**官方文档：** https://code.claude.com/docs/zh-CN/skills

**核心要求：**
- Skill 目录结构：`.claude/skills/<skill-name>/SKILL.md`
- YAML frontmatter 配置
- 调用控制（disable-model-invocation, user-invocable）
- 动态上下文注入支持

### 1.2 Agent Skills 开放标准

**官方网站：** https://agentskills.io

**核心要求：**
- 跨平台 Skill 格式
- 工具定义规范
- 命令注册机制
- 兼容性指南


**符合性检查：**
- [x] Skill 目录位于 `.claude/skills/` 下
- [x] 包含必需的 `SKILL.md` 文件
- [x] 目录名称与 skill 名称一致
- [x] 支持文件拆分（超过 500 行时）

### 2.2 SKILL.md Frontmatter

```yaml
---
name: 115-skills
description: 115 网盘智能管理：扫码登录、文件浏览、搜索、转存、离线下载、智能整理
argument-hint: [action] [target] [options]
disable-model-invocation: true
user-invocable: true
allowed-tools: Read, Grep, Glob, Bash
---
```

**符合性检查：**
- [x] `name` 字段：小写 + 连字符
- [x] `description` 字段：描述触发场景
- [x] `argument-hint` 字段：参数提示
- [x] `disable-model-invocation` 字段：高危操作控制
- [x] `user-invocable` 字段：用户可调用
- [x] `allowed-tools` 字段：最小权限原则
