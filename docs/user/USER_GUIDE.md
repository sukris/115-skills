## 快速开始

### 1. 安装 Skill

```bash
npx clawhub --workdir ~/.openclaw --dir skills install 115-skills
```

### 2. 登录 115


## 常见问题

### Q1: 扫码后没有反应？

**A:** 请检查：
1. 手机网络是否正常
2. 115 APP 是否为最新版本
3. 二维码是否过期（5 分钟）

如果问题持续，重新发送"登录 115"获取新二维码。

### Q2: 转存失败怎么办？

**A:** 可能原因：
1. 分享已失效
2. 提取码错误
3. 存储空间不足

解决方法：
- 确认分享链接有效
- 检查提取码是否正确
- 清理空间后重试

### Q3: 离线下载速度慢？

**A:** 离线下载速度取决于：
1. 资源热度（种子数量）
2. 115 服务器负载
3. 你的 VIP 等级

建议：
- 选择热门资源
- 开通 VIP 加速

### Q4: 如何退出登录？

**A:** 目前 Cookie 会自动管理，90 天后自动过期。如需立即退出，删除 `~/.openclaw/115-cookie.json` 文件。

### Q5: 支持哪些文件格式？

**A:** 支持所有文件格式的上传、下载、转存。智能整理支持以下分类：
- 📷 图片
- 🎬 视频
- 🎵 音乐
- 📄 文档
- 📦 压缩包
- 💻 代码
- 📱 应用
- 📚 书籍
- 其他

## 最佳实践

### 1. 文件命名规范

**推荐：**
```
[日期]_[项目]_[类型].[扩展名]
例：20260314_115Skill_设计稿.psd
```

### 2. 目录结构建议

```
/我的资源/
├── 📷 图片/
├── 🎬 视频/
├── 🎵 音乐/
├── 📄 文档/
├── 📦 压缩包/
└── 💻 代码/
```

### 3. 定期清理

建议每月执行一次：
```
你：清理建议
你：整理文件
```

### 4. 批量操作技巧

转存多个文件时，使用"一键转存全部"功能，避免逐个操作。

### 5. 离线下载优化

- 优先使用磁力链接（成功率高）
- 大文件使用离线下载（节省带宽）
- 定期清理已完成任务
